1. Banking System
- 이 프로젝트는 은행 계좌 관리 시스템의 일부분으로, 계좌 생성, 삭제, 조회, 잔액 사용, 잔액 사용 취소, 거래 확인 등의 기능을 제공합니다.

2. 핵심 기능
- 계좌 생성
- 계좌 삭제
- 계좌 조회
- 잔액 사용
- 잔액 사용 취소
- 거래 확인

3. 사용 기술
- Java
- Spring Framework
- Redis
- JPA
- Hibernate

4. 주의사항
- 계좌는 한 사용자 당 최대 10개까지 생성할 수 있습니다.
- 거래 금액은 잔액을 초과할 수 없습니다.
- 거래 취소는 거래 일자 기준으로
